## Rodando o projeto

### Clonando repositório

```bash
git clone https://github.com/Matheusvicentesn/labCAR.git .
```

### Instalando dependências

```bash
npm install
```

## Iniciando aplicação

```bash
npm run start:dev
```

## Acessando documentação

```bash
# Podendo ser acessada via navegador e outros clientes, ex: Postman,Insomnia
localhost:3000/api#/
```
